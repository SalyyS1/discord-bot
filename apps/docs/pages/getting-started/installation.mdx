# Installation

Add SylaBot to your Discord server in just a few steps.

## Step 1: Invite the Bot

Click the button below to add SylaBot to your server:

[**Add SylaBot to Discord**](https://discord.com/oauth2/authorize?client_id=1462790510883110965&permissions=8&scope=bot%20applications.commands)

## Step 2: Grant Permissions

SylaBot requires the following permissions:
- **Administrator** (recommended) - For full functionality
- Or individual permissions for specific features

## Step 3: Access the Dashboard

1. Go to [dashboard.sylabot.io](https://dashboard.sylabot.io)
2. Sign in with Discord
3. Select your server from the dropdown

## What's Next?

Continue to [First Setup](/getting-started/first-setup) to configure your bot.
