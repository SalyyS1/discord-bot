# First Setup

Configure SylaBot for your server after installation.

## Access the Dashboard

1. Visit [dashboard.sylabot.io](https://dashboard.sylabot.io)
2. Click "Sign in with Discord"
3. Select your server from the dropdown

## Essential Settings

### Welcome Messages
Set up automatic welcome messages for new members:

1. Go to **Features** → **Welcome**
2. Enable welcome messages
3. Select a channel
4. Customize the message template

### Leveling System
Enable XP and leveling:

1. Go to **Features** → **Leveling**
2. Toggle "Enable Leveling"
3. Set XP rate and multipliers
4. Configure level-up announcements

### Moderation
Set up basic moderation:

1. Go to **Features** → **Moderation**
2. Configure auto-mod rules
3. Set log channel for mod actions

## Testing

After setup, test by:
- Having a member join (welcome message)
- Sending messages (XP gain)
- Using `/rank` command
