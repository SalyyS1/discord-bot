# Giveaways

Host engaging giveaways for your community.

## Overview

Create giveaways with:
- Timer-based entries
- Entry requirements
- Multiple winners
- Bonus entries

## Setup

1. Go to **Dashboard** → **Features** → **Giveaways**
2. Click "Create Giveaway"

## Creating a Giveaway

| Field | Description |
|-------|-------------|
| Prize | What winners receive |
| Duration | How long to run |
| Winners | Number of winners |
| Channel | Where to post |

## Commands

| Command | Description |
|---------|-------------|
| `/giveaway create` | Create new giveaway |
| `/giveaway end` | End early |
| `/giveaway reroll` | Pick new winner |
