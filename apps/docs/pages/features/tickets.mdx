# Ticket System

Professional support ticket management for your server.

## Overview

Create a streamlined support experience with:
- Custom ticket panels
- Multiple categories
- Staff assignment
- Transcript archives

## Setup

1. Go to **Dashboard** → **Features** → **Tickets**
2. Enable the ticket system
3. Create a ticket panel
4. Deploy to a channel

## Creating a Panel

1. Click "Create Panel"
2. Configure:
   - Panel title and description
   - Button text and color
   - Ticket category channel
   - Support roles

## Commands

| Command | Description |
|---------|-------------|
| `/ticket close` | Close current ticket |
| `/ticket add @user` | Add user to ticket |
| `/ticket remove @user` | Remove from ticket |
| `/ticket claim` | Claim ticket (staff) |

## Ticket Flow

```
User clicks button → Ticket channel created → Staff notified → 
Resolution → Close → Transcript saved
```
