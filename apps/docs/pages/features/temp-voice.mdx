# Temp Voice Channels

Dynamic voice channels created on demand.

## Overview

Users can create temporary voice channels that:
- Auto-delete when empty
- Are fully customizable
- Support user limits
- Have owner controls

## Setup

1. Go to **Dashboard** → **Features** → **Temp Voice**
2. Enable the feature
3. Set a "creator" channel
4. Configure defaults

## How It Works

1. User joins creator channel
2. Bot creates new private channel
3. User becomes channel owner
4. Channel deletes when empty

## Commands

| Command | Description |
|---------|-------------|
| `/voice name` | Rename your channel |
| `/voice limit` | Set user limit |
| `/voice lock` | Lock channel |
| `/voice permit @user` | Allow user in |
