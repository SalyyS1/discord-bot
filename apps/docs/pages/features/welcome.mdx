# Welcome Messages

Greet new members with customizable messages.

## Overview

- Custom text and embeds
- Variable placeholders
- Images and GIFs
- Role assignment

|----------|--------|

| `{username}` | Username |
| `{server}` | Server name |
| `{memberCount}` | Member count |
