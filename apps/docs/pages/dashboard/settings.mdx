# Dashboard Settings

Configure general bot settings.

## General Settings

| Setting | Description |
|---------|-------------|
| Bot Prefix | Command prefix (legacy) |
| Language | Bot response language |
| Timezone | Server timezone |

## Logging

Configure what gets logged:
- Moderation actions
- Message edits/deletes
- Member join/leave
- Voice state changes

## Permissions

Set who can access specific features via roles.
