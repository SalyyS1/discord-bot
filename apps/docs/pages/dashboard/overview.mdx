# Dashboard Overview

Manage SylaBot settings through the web dashboard.

## Accessing the Dashboard

1. Visit [dashboard.sylabot.io](https://dashboard.sylabot.io)
2. Sign in with Discord
3. Select your server

## Navigation

- **Home** - Server overview and stats
- **Features** - Configure bot features
- **Settings** - General server settings
- **Logs** - View activity logs

## Requirements

You need **Manage Server** permission in Discord to access the dashboard.
