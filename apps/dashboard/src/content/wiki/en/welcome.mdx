# Welcome Messages

Create engaging welcome and goodbye messages to greet your community members.

## Overview

The welcome system sends customized messages when members join or leave your server. Include images, embeds, and personalized text to make a great first impression.

## Basic Setup

### Step 1: Enable Welcome Messages

1. Navigate to **Dashboard** â†’ **Welcome**
2. Toggle **"Enable Welcome Messages"**
3. Select welcome channel
4. Configure message settings
5. Save changes

### Step 2: Enable Goodbye Messages

1. Toggle **"Enable Goodbye Messages"**
2. Select goodbye channel (can be same as welcome)
3. Configure goodbye message
4. Save changes

### Available Variables

Use these placeholders in messages:

- `{{user}}` - Mentions the new member
- `{{username}}` - Member's username (no mention)
- `{{server}}` - Server name
- `{{memberCount}}` - Total member count
- `{{date}}` - Current date
- `{{time}}` - Current time

## Customization Options

### Message Timing

Configure when messages send:
- **Immediate** - Right when member joins
- **Delayed** - Wait X seconds/minutes
- **Conditional** - Only if member stays X minutes

### Member Filtering

Control who gets welcome messages:
- Skip bots
- Skip rejoining members (within 24 hours)
- Skip members below account age threshold
- Skip members with suspicious activity

### Channel Permissions

Ensure proper permissions:
- Bot can **Send Messages** in welcome channel
- Bot can **Embed Links** for embeds
- Bot can **Attach Files** for images
- Bot can **Manage Roles** for auto-roles

Track welcome system performance:

- **Total Welcomes** - All-time welcome messages sent

## Best Practices

1. **Keep It Friendly** - Warm, inviting tone
2. **Provide Direction** - Link to rules, roles, and key channels
3. **Use Images** - Visual appeal increases engagement
4. **Test Thoroughly** - Verify variables and formatting work
5. **Update Regularly** - Keep info current (events, updates)
6. **Consider Mobile** - Ensure messages look good on mobile
7. **Avoid Spam** - Don't make messages too long or mention too many roles

## Troubleshooting

**Messages Not Sending**: Check bot permissions and channel selection.

**Variables Not Working**: Ensure correct syntax with `{{variable}}`.

**Images Not Showing**: Verify bot has Attach Files permission.

**DMs Not Sending**: User likely has DMs disabled (this is normal).
