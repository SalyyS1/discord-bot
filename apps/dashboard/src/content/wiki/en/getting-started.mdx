# Getting Started with SylaBot

Welcome to SylaBot! This guide will help you get started with setting up and configuring your Discord bot dashboard.

## Prerequisites

Before you begin, make sure you have:

- **Discord Server** - You need administrator permissions
- **SylaBot Invited** - Add the bot to your server
- **Discord Account** - Required for dashboard access

## Step 1: Add SylaBot to Your Server

1. Click the **"Add to Server"** button on the landing page
2. Select your Discord server from the dropdown
3. Review and authorize the required permissions
4. Complete the verification if prompted

## Step 2: Access the Dashboard

1. Click **"Login"** on the navigation bar
2. Authorize with your Discord account
3. Select your server from the server list
4. You'll be redirected to the dashboard overview

## Step 3: Configure Basic Settings

### Select Your Server

Use the server selector in the sidebar to switch between your servers. The dashboard will load settings specific to the selected server.

### Overview Page

The overview page displays:

- **Member Statistics** - Total tracked members and growth
- **Activity Metrics** - Messages, tickets, and giveaways
- **Feature Status** - Quick view of enabled features
- **Recent Activity** - Latest moderation actions
